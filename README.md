## What is Numly?

Numly is a free AI-powered calculator for iOS that understands natural language. Instead of tapping buttons, just type what you mean:

- **"split $120 between 5 people with 18% tip"** → $28.32 each
- **"what's 30% off $149"** → $104.30
- **"15% of my $2,400 salary"** → $360
